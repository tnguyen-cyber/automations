## Automation Production Systems

#### SMB – Media Staging Pipeline
Pipeline for transforming raw media into deterministic, CDN-ready assets.

**Highlights:**
- CSV-driven SKU mapping
- Media normalization (JPG/MOV → standardized formats)
- Deterministic file naming (`image.jpg`, `video.mp4`)
- Structured output for CDN compatibility

➡️ [View Project](./smb-media-pipeline/)
